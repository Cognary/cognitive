# Conformance Center

Conformance defines how strictly a runtime implements Cognitive 2.2.7 behavior.

## Scope

- Envelope contract (`ok/meta/data/error/partial_data`)
- Tier policy behavior (`exec/decision/exploration`)
- Validation behavior (schema and policy checks)
- Compatibility behavior (v2.1 to v2.2 conversion)
