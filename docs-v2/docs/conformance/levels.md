# Conformance Levels

## Level 1

Baseline envelope and validation correctness.

## Level 2

Tier-aware policy behavior and strictness controls.

## Level 3

Advanced orchestration and context/composition consistency.

## Claiming Conformance

A runtime should only claim a level after passing all required vectors for that level.
