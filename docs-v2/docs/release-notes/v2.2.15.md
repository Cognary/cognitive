# v2.2.15

Release date: 2026-03-09

## Highlights

- Runtime normalization: output canonicalization now respects schema hints such as `x-cognitive-string-normalize` and `x-cognitive-unordered`, reducing contract drift before validation.
- PR Risk Gate: the repository now ships an official `pr-risk-gate` module and a copy-paste GitHub Actions template for blocking risky pull requests.
- Evidence page: benchmark results comparing `raw-text`, `raw-schema`, `cognitive-core`, and `cognitive-standard` are now published in docs, with Gemini and MiniMax runs both showing stable contract execution for the Cognitive paths.
