# v2.2.8

Release date: 2026-02-07

## Highlights

- Fix: `npx cogn@2.2.7` failed on Node 24+ due to package `"exports"` + ESM/CJS interop. `cogn@2.2.8` now spawns the CLI entry instead of `require()`-ing `dist/cli.js`.
- Hardening: registry index client supports explicit timeout and payload caps:
  - CLI: `--registry-timeout-ms`, `--registry-max-bytes`
  - ENV: `COGNITIVE_REGISTRY_TIMEOUT_MS`, `COGNITIVE_REGISTRY_MAX_BYTES`
- Hardening: `cog registry verify --remote` supports bounded concurrency via `--concurrency` (max 8).

## Distribution

- npm: `cognitive-modules-cli@2.2.8` and `cogn@2.2.8`
- Registry tarballs (GitHub Release assets): `*-2.2.8.tar.gz`
- PyPI (legacy): `cognitive-modules@2.2.3`
