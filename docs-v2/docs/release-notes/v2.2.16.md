# v2.2.16

Release date: 2026-03-09

## Highlights

- Loader fix: module-local schema references now work during runtime validation, even when `schema.json` keeps shared definitions under root `$defs` or `definitions`.
- PR Risk Gate: the official `pr-risk-gate` module now passes end-to-end smoke tests again, including the published Gemini gate example.
- Runtime hardening: validation failures now reflect real contract problems instead of broken internal schema resolution.
