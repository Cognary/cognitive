# v2.2.14

Release date: 2026-03-08

## Highlights

- Release engineering: `release:check` now builds `docs-v2` before packaging, so broken MDX/docs links fail locally before publish.
- Packaging: `packages/cogn` now reuses the primary runtime gate, preventing alias-only publishes when the main runtime is not release-ready.
- Provider hardening: Anthropic streaming usage aggregation now preserves prompt token counts when later deltas omit earlier fields.
- Test coverage: stable provider request-shaping coverage expanded across Anthropic, OpenAI, MiniMax, and Qwen paths.
