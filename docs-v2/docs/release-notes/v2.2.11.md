# v2.2.11

Release date: 2026-02-07

## Highlights

- Fix (Gemini): improve `responseSchema` compatibility by dropping non-string `enum`/`const` constraints (Gemini rejects boolean enums).
- UX: safer `npx` argument handling for `--provider/--model` in `core` commands (avoid "flag position" surprises).
- Provider: Moonshot/Kimi model constraints are enforced where required (for example, some models only accept specific temperature settings).
