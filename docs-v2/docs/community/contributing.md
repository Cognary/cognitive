# Contributing

## How to Contribute

- Report issues with reproducible steps.
- Propose spec/runtime changes with clear compatibility impact.
- Add tests for behavior changes.

## Quality Bar

- Keep contracts explicit.
- Preserve backward compatibility unless clearly versioned.
- Update docs and changelog with every user-visible change.
