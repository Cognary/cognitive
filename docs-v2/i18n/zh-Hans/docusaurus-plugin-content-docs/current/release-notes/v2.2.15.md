# v2.2.15

发布日期：2026-03-09

## 重点

- 运行时归一化：输出 canonicalization 现在会读取 schema hints（如 `x-cognitive-string-normalize`、`x-cognitive-unordered`），在校验前减少合同漂移。
- PR 风险门禁：仓库现在内置官方 `pr-risk-gate` 模块，并附带可直接复制的 GitHub Actions 模板，用于阻断高风险 PR。
- 证据页：文档已公开 `raw-text`、`raw-schema`、`cognitive-core`、`cognitive-standard` 的 benchmark 对照结果；Gemini 与 MiniMax 两组运行都显示 Cognitive 路径具备稳定合同执行能力。
