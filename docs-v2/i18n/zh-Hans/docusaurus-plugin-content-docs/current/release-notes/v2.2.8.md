# v2.2.8

发布日期：2026-02-07

## 重点

- 修复：`npx cogn@2.2.7` 在 Node 24+ 下会因包 `"exports"` + ESM/CJS 互操作失败而报错。`cogn@2.2.8` 改为 spawn 执行 CLI entry，不再 `require()` `dist/cli.js`。
- 加固：registry index 拉取支持显式的超时与最大 payload 上限：
  - CLI：`--registry-timeout-ms`、`--registry-max-bytes`
  - 环境变量：`COGNITIVE_REGISTRY_TIMEOUT_MS`、`COGNITIVE_REGISTRY_MAX_BYTES`
- 加固：`cog registry verify --remote` 支持 `--concurrency` 并发门禁（最大 8）。

## 发布渠道

- npm：`cognitive-modules-cli@2.2.8` 与 `cogn@2.2.8`
- registry tarball（GitHub Release 资产）：`*-2.2.8.tar.gz`
