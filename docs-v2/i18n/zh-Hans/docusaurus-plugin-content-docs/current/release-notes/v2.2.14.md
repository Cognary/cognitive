# v2.2.14

发布日期：2026-03-08

## 重点

- 发布工程：`release:check` 现在会先构建 `docs-v2`，这样 MDX/文档链接错误会在本地发布前直接失败。
- 打包门禁：`packages/cogn` 现在复用主运行时门禁，避免主运行时未达发布条件时单独发布 alias 包。
- Provider 加固：Anthropic 流式 usage 聚合现在会保留 prompt token 统计，不再因为后续 delta 缺少字段而把先前计数覆盖掉。
- 测试覆盖：补齐 Anthropic、OpenAI、MiniMax、Qwen 的稳定 provider 请求整形测试。
