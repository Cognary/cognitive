# v2.2.7

发布日期：2026-02-06

## 重点

- Node 运行时在路径安全与校验一致性上完成加固。
- 跨接口输出行为围绕 envelope 语义进一步统一。
- CLI/MCP/HTTP 与 composition 行为契约更可预测。

## 口径澄清

`compose --trace` 仍是 trace/debug 包装输出模式。
该模式有意区别于非 trace 的纯 envelope 输出。

- npm：`cognitive-modules-cli@2.2.7` 与 `cogn@2.2.7`
- registry tarball（GitHub Release 资产）：`*-2.2.7.tar.gz`
